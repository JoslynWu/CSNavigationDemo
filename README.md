#                NavigationDemo


## 概要
这是一个关于navigationBar的Demo。  

## 介绍
### Demo列举了四种NavigationBar的样式 
![](/media/nav_list.png)
1. 系统默认样式。（便于对比）
2. 系统样式下调整了颜色。
![](/media/nav_system_config.png)
3. 自定义导航栏，随滚动渐变透明度。
![](/media/nav_custom_shade.png)
4. 整个headerView随滚动渐变，最后呈现出导航栏。
[](/media/nav_custom_shadePull.png)
